How to fix error 80073712 when installing Windows 8.1 update KB2919355 (PowerShell)

Introduction
This script shows how to fix error 80073712 when installing Windows 8.1 update KB2919355.

Scenarios
After Windows 8.1 update released, some customer come across an issue that they will get an error when they install KB2919355. This script is trying to fix that issue. This script is trying to fix that issue.

Script

Step 1: Right Click the script and select Run with PowerShell.

Step 2: Then there will be a dialog. After it finishes, you can run Windows Update and install KB8919355 again.

Here are some code snippets for your references

- PowerShell code snippet -
 
If($version.Caption -match "Microsoft Windows 8")
{
    #64Bit
    If($version.OSArchitecture -eq "64-bit")
    {
        if($item)
        {
            Dism /online /remove-package /packagename:Package_for_KB2919355~31bf3856ad364e35~amd64~~6.3.1.14
        }
    }
    #32Bit
    ElseIf($version.OSArchitecture -eq "64-bit")
    {
        if($item)
        {
            Dism /online /remove-package /packagename:Package_for_KB2919355~31bf3856ad364e35~x86~~6.3.1.14
        }
    }
 
    Dism /Online /Cleanup-image /Restorehealth
 
    Dism /online /cleanup-image /startcomponentcleanup
 
    Write-Host "Operation Done. Run Windows Update again and try to re-install KB2919355. If this script still fail to resolve your error, please contact Microsoft Support."
}
 
- end -
